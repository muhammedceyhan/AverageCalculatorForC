# Average Calculator for C

A simple and efficient C program designed to calculate the running average of a sequence of numbers entered by the user.

## ğŸ“ Description

This application runs in an infinite loop, continuously prompting the user to input integer values. For each entered number, the program:
1. Adds the number to a running total.
2. Increments the count of entries.
3. Instantly calculates and displays the new average with 2 decimal precision.

It's a straightforward tool useful for quick statistical calculations or as a learning example for basic C programming concepts like loops, input/output, and type casting.
